## Reporting Security Issues

### Security Vulnerabilities

**DO NOT** report security issues publicly on GitHub Issues.

**Email**: security@neonexcore.dev

**Include**:
- Vulnerability description
- Steps to reproduce
- Affected versions
- Proof of concept (if available)
- Your contact information

**Response Time**: Within 24 hours

**Process**:
1. We acknowledge receipt within 24 hours
2. We investigate and validate the issue
3. We develop a fix
4. We notify you when patch is ready
5. We coordinate disclosure timeline
6. We release patch and advisory
