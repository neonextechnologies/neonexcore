## Issue Tracking

### Bug Reports

Found a bug? [Create an issue](https://github.com/neophp/neonexcore/issues/new?template=bug_report.md)

Include:
- Description
- Steps to reproduce
- Expected behavior
- Actual behavior
- Environment (OS, Go version)
- Code samples

### Feature Requests

Have an idea? [Request a feature](https://github.com/neophp/neonexcore/issues/new?template=feature_request.md)

Include:
- Use case
- Proposed solution
- Alternative solutions
- Additional context
